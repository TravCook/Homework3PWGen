### RANDOM PASSWORD GENERATOR ###
  This web application gets user input through a series of prompts, and then randomly generates a password using th given criteria. It asks the user if they would like to include up to 4 different criteria (lowercase, uppercase, numbers, and special charactes) then it uses random number generation to pick a different output each time. It then adds all of the outputs into a single array, and fully joins it and prints it right into the webpage.
